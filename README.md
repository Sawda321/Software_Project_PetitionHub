# 📋 PetitionHub - Software Engineering Project

PetitionHub is an online platform where users can create petitions for various social, local, or institutional causes, and others can sign them to show their support and solidarity.

## 🚀 Features
* **User Authentication:** Secure sign-up and login system for users.
* **Create Petitions:** Users can launch new petitions or campaigns easily.
* **Trending Section:** A dedicated area to showcase the most popular and active petitions.
* **Clean UI:** User-friendly and responsive design for a smooth experience.

## 🛠️ Technology Stack
* **Frontend:** HTML5, CSS3, JavaScript

## 💻 How to Run the Project
1. Clone or download this repository to your local machine.
2. Open the project folder in your preferred code editor (e.g., VS Code).
3. Open the `index.html` (or your main landing page file) in any modern web browser.
